# iOS Platform

The `ios/App/` Xcode project is generated by Capacitor on first setup:

```bash
npm run ios:setup
```

## Supported devices

Optimized for iPhone 12 through iPhone 17 Pro Max:

| Device | Width (pt) |
|--------|------------|
| iPhone 12 / 13 / 14 | 390 |
| iPhone 12–14 Pro Max | 428 |
| iPhone 15 / 16 | 393 |
| iPhone 15–16 Pro Max | 430 |
| iPhone 16 Pro | 402 |
| iPhone 17 Pro Max | ~440 |

## Permissions (Info.plist)

Applied automatically by `scripts/setup-ios.sh`:

- `NSCameraUsageDescription`
- `NSPhotoLibraryUsageDescription`
- `NSPhotoLibraryAddUsageDescription`

## Manual Xcode steps

1. Open `ios/App/App.xcworkspace`
2. Set Team & Signing
3. Set deployment target iOS 14.0+
4. Product → Archive → Distribute

Or use GitHub Actions (see root README).
